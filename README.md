# Death Road
2D 게임 프로젝트, Earn To Die 플래쉬 게임의 모작

작업 설명

조작법 WASD → w 가속 s 감속 a,d 회전. 조작법 Space → 부스터 사용 

1 키 → 이것은 치트키이다. 1000 의 돈이 추가된다.

하위 작업
- [ ]  차량 업그레이드
    - [ ]  Gun 업그레이드 → 7초 마다 사거리 안에 들어오는 좀비를 파괴 시킨다. 총 1 레벨 만 존재,
    - [ ]  Fuel 업그레이드 → 연료의 총 량을 늘려준다. 총 10레벨, 각 레벨 마다 총 연료의 량이 다르다.
    - [ ]  Booster 업그레이드 → 1 레벨 때는 부스터를 얻는다, 2 ~3 레벨 부터는 부스터의 량을 늘려준다.
    - [ ]  Engine 업그레이드 → 차량의 속도를 올려준다 총 3레벨.
    - [ ]  Bumper 업그레이드 → 차량의 범퍼에 좀비를 파괴 시키는 부품을 달아준다, 좀비 나 파괴 가능한 오브젝트와 충돌했을때 바로 파괴시켜준다. 총 1레벨
- [ ]  차량 구매
    - [ ]  jeep 차 구매 : 3000 머니가 필요하다.
    - [ ]  truck 차 구매: 6000 머니가 필요하다.
- [ ]  총 거리
    - [ ]  상단 에 Slider 를 통해 현재 위치 와 목표 위치 까지의 거리 를 나타냈다.
    - [ ]  또 현재 까지 달려온 거리를 텍스트로 나타냈다.
- [ ]  점수 와 머니
    - [ ]  현재 까지 달려온 거리 를 돈으로 환산한다.  float 를 int 로 바꿔소 소수점을 없앴다.
    - [ ]  좀비 킬 수 * 5 만큼의 돈을 얻는다.
    - [ ]  좀비 파괴 수 * 10 만큼의 돈을 얻는다.
- [ ]  장애물 충돌 과 파괴
    - [ ]  장애물 과 의 충돌에서 차량의 속도가 파괴 의 필요한 속도 보다 이상이면 장애물들이 파괴된다
- [ ]  좀비의 충돌 과 사망 그리고 파괴
    - [ ]  차량과 충돌을 하면, 차량의 속도 와 비교해 좀비의 죽음을 판단한다.
    - [ ]  좀비가 파괴 될 시 UI 로 간단한 텍스트 와 시간을 0.5 로 바꾸어 파괴를 알렸다.
    - [ ]  죽으면 래깅돌 상태가 된다, 파괴 될 정도로 속도가 놓으면 파괴가 된다.
- [ ]  좀비 의 종류 와 각 특수 행동
    - [ ]  일반 좀비 : 그냥 일반 적인 좀비
    - [ ]  빠른 좀비 : 일반 좀비 + 빠른 속도
    - [ ]  메달리는 좀비 : 차량의 향해 달려와 메달림, 일정 속도가 되야 떨어진다.
    - [ ]  군인 좀비 : 일정 구간에 들어오면 슈류탄을 던진다. 슈류탄은 3 초 있다가 터진다.
    - [ ]  탱크 좀비 : 그냥 엄청 무겁고, 딴딴 하다. 느리다.
- [ ]  차량 고스트 표시
    - [ ]  차량이 움직이면서, 각 위치와 회전을 기록한다, 그리고 재생한다.
    - [ ]  각 차량 에 맞는 이미지, 각 부품의 유뮤에 따른 기록
