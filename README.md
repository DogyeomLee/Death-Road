Death Road
2D Physics-Based Vehicle Action Project

본 프로젝트는 플래시 게임 'Earn to Die'를 모티브로 한 2D 횡스크롤 액션 게임입니다. 물리 엔진을 활용한 차량 조작과 좀비 파괴 메커니즘, 그리고 고스트 시스템을 통한 기록 재현을 핵심으로 합니다.

핵심 시스템 (Core Systems)
1. 차량 물리 및 업그레이드
  * 물리 조작: 가속, 감속 및 공중 회전(W,A,S,D)을 리지드바디(Rigidbody2D)의 토크와 힘으로 제어합니다.
  * 데이터 주도 업그레이드: 각 부품의 레벨별 수치를 관리하며, 엔진 성능, 연료량, 부스터 출력 등을 강화할 수 있습니다.  
  * 차량 교체: 획득한 재화를 소모하여 더 높은 기본 성능을 가진 상위 차량(Jeep, Truck)을 구매하는 진행 구조를 갖췄습니다.

2. 물리 기반 파괴 및 타격감
  * 속도 기반 파괴: 충돌 시 차량의 속도가 일정 수치 이상일 때만 장애물과 좀비가 파괴되는 로직을 구현했습니다.
  * 타격감 연출: 좀비 파괴 시 프레임 속도를 일시적으로 늦추는(Time Scale 0.5) 히트 스탑 기법과 텍스트 피드백을 활용했습니다.
  * 래그돌 시스템: 좀비가 사망하거나 파괴될 때 물리 엔진이 적용된 래그돌 상태로 전환되어 자연스러운 시체 처리를 보여줍니다.

3. 고스트 리플레이 시스템
  * 주행 기록 및 재생: 차량의 위치, 회전값, 장착된 부품 정보를 실시간으로 기록한 뒤, 다음 플레이 시 고스트 차량으로 재생하여 이전 기록과 대조할 수 있게 했습니다.
  * 좀비 개체 및 특성 (Enemies)
    1. 일반 좀비: 표준적인 체력과 물리 상호작용을 가진 개체입니다.
    2. 빠른 좀비: 높은 이동 속도로 차량을 추격하며 위협합니다.
    3. 매달리는 좀비: 차량에 부딪히면 떨어지지 않고 매달려 물리적인 저항(무게)을 추가하며, 탈출을 위해 가속이 필요합니다.
    4. 군인 좀비: 일정 범위 진입 시 3초 후 폭발하는 수류탄을 투척하여 범위 피해를 줍니다.
    5. 탱크 좀비: 거대한 질량과 높은 방어력을 가진 중량급 장애물입니다.

업그레이드 및 기능 상세
  1. Gun: 7초 간격으로 사거리 내 적을 자동 사격하여 제거합니다.
  2. Fuel: 총 10레벨로 구성되며, 레벨에 따라 연료 최대치를 늘려 주행 시간을 확보합니다.
  3. Booster: 1레벨에서 기능을 해금하고, 이후 레벨업을 통해 부스터 게이지 용량을 확장합니다.
  4. Engine: 차량의 가속력과 최대 속도를 결정하는 핵심 엔진 출력을 강화합니다.
  5. Bumper: 충돌 시 감속을 방지하고 좀비를 즉시 파괴하는 보조 부품을 장착합니다.

조작 및 UI 기능
  1. 기본 조작: W(가속), S(감속), A/D(공중 회전), Space(부스터)
  2. 치트 기능: 테스트 편의를 위한 1번 키 머니 추가 기능 (1,000 머니)  
  3. 진행도 표시: 상단 슬라이더 UI를 통해 현재 위치와 목표 지점까지의 남은 거리를 실시간으로 시각화했습니다.
  4. 재화 환산: 주행 거리(Float)를 정수(Int)로 변환하고 좀비 처치/파괴 수에 따른 보너스를 합산하여 최종 머니를 산출합니다.

기술적 성과 (Technical Insights)
  1. 데이터 정합성: 부동 소수점 오차를 고려한 거리 계산 및 형변환을 통해 정확한 재화 보상 시스템을 구축했습니다.
  2.물리 최적화: 다수의 좀비와 장애물이 파괴되는 과정에서도 안정적인 프레임을 유지하기 위해 물리 판정을 최적화했습니다.
